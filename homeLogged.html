Welcome to SpellForge!

SpellForge is an open source website that allows users to create spell builds in order to manage material components in Dungeons & Dragons™ games. This project began as a project for Worcester Polytechnic Institute's Human-Computer Interaction course in C-Term of 2024.

Functions
  • Create spell builds to use in games.
  • Manage material components.
  • Add and remove spells from builds.
